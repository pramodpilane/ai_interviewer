# 🤖 AI-Powered CLI Technical Interviewer

A LangChain-based command-line app that simulates a technical interviewer, dynamically asks questions, evaluates responses, and gives feedback.

---

## 🚀 Features

- Dynamic question generation (3–5 questions)
- Real-time evaluation using LLM
- Summary and score at the end
- Multi-line answer support

---

## 🧠 Technologies Used

| Package               | Purpose                                                   |
|-----------------------|-----------------------------------------------------------|
| **langchain**         | High-level framework to compose LLM applications          |
| **langchain-core**    | Core primitives and types used by LangChain               |
| **langchain-openai**  | Integrates OpenAI models (e.g., GPT-3.5, GPT-4) with LangChain |
| **python-dotenv**     | Loads environment variables from `.env` file              |
